Ukratko o meni i o projektu.

Pozdrav,
moje ime je Dejan Petrović i ja sam autor ove stranice.

Ovu stranicu sam napravio na Bootcamp-u u toku školovanja za java programera koju je organizovala Bild organizacija. To školovanje pohadja, uključujući i mene, oko dvadesetak učenika. Bilo je to u sklopu sedmičnog web projekta, a zadatak je bio da se sastavi web stranica po izboru kako bi se izvježbale web tehnologije. Iskreno nisam imao neku posebnu ideju za temu web stranice, ali čim sam saznao da treba da sastavimo web stranicu znao sam da ću sastaviti nesto podržano Bootstrap projektom. Ova web stranica je zasnovana na Bootstrap-ovom templejtu Cover. Svrha stranice je da sluzi kao muzička inspiracija. Plejlista posjeduje strogo probranu i odabranu muziku od veoma kvalitetnih izvodjača.

Više o Bild organizaciji mozete pročitati na njihovoj web stranici mozemo.ba.

Plejlista je napravljena uz pomoć Bootstrap projekta, autor @AonoZan.
